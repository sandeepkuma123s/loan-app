# loan-app
# 🏦 Loan Prediction Web App

A Machine Learning based web application that predicts whether a loan will be **Approved ✅ or Rejected ❌** based on user inputs.

## 📌 Project Overview

This project uses a **Machine Learning model** trained on real loan dataset to help banks or users predict loan approval status.

Users enter details like:

* Applicant Income
* Loan Amount
* Credit History
* Employment & Property details

👉 The model predicts the result instantly.

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/YOUR-USERNAME/loan-app.git
cd loan-app
```

---

### 2️⃣ Create virtual environment

```
python -m venv venv
venv\Scripts\activate
```

---

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

---

### 4️⃣ Run the app

```
python app.py
```

---

### 5️⃣ Open in browser

```
http://127.0.0.1:5000
```
